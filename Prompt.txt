You are a professional front-end developer.  
Create a modern, responsive multi-page website for a Social Media Ads Agency called **"VarniElevate Elevate"**.  
The website should look like a premium digital marketing / performance ads agency site.

=============================
GENERAL REQUIREMENTS
=============================

1. Tech Stack:
   - Use HTML, CSS, and JavaScript only (no frameworks required).
   - Make the design fully responsive (mobile, tablet, desktop).

2. Design Style:
   - Clean, minimal, professional, agency-style.
   - Use sections with enough white space.
   - Rounded buttons, subtle hover effects.
   - Use icons (can be from any icon library or simple placeholders).

3. Color Palette (Brand Colors):
   - Primary: Navy Blue – #0A1A2F
   - Accent: Sky Blue – #00AEEF
   - Background: White – #FFFFFF
   - Neutral: Light Grey – #F5F5F5
   - Text: Dark Grey / almost black

4. Typography:
   - Headings font: Poppins or Montserrat (or similar clean sans-serif).
   - Body font: Inter or Open Sans (or similar).
   - Heading sizes should be bold and clear.
   - Body text must be readable with good line spacing.

5. Global Layout:
   - Sticky header with logo and navigation.
   - Smooth scroll to sections on the same page where relevant.
   - CTA (Call-To-Action) buttons should stand out using the accent color (#00AEEF).
   - Add consistent padding/margins throughout for a premium feel.

6. Pages to Create:
   - Home
   - Services
   - Case Studies
   - About
   - Contact
   (Pricing can be a section on Home or Services; you can decide best UX-wise.)

Use the following content and structure in detail.
